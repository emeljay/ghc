Activity 1

Set up

  1. Download workshop materials here, if you haven't already.
  2. Open the workshop materials folder and navigate to the website/ directory.
  3. Open index.html in Chrome. NOTE: If you already feel comfortable using a screen reader on your machine, open index_no_screenreader.html instead.

Exercises

  1. Use the "Heading" button on the screen reader to jump to all the heads in the app. Are you able to use this button to jump to the very first site title heading on the page?
  2. Try using the keyboard (tab and enter key) to buy an item.
  3. Click the first "buy" button using the mouse. You'll notice a dialog appear. Without looking at the screen, did you receive any indication from the screen reader that a dialog popped up? Do you know what the dialog is for?
  4. Using the keyboard, try to tab to the first input field in the dialog.
  5. Once you've tabbed to the input fields in the dialog, try filling out the fields. Looking at the screen, you can determine that the input fields are for quantity, name, and email. Without looking at the screen, can you figure out what each input field is for?
  6. Try navigating to the "X" button on the dialog.
  7. Try selecting the "submit" button.
